# NYC Taxi Duration Prediction: End-to-End MLOps Pipeline

A production-ready MLOps project to predict taxi trip durations in NYC. This project demonstrates a complete lifecycle from raw data ingestion to containerized deployment.

### Key Features:
* **Data Pipeline:** Automated ingestion and preprocessing of NYC Parquet data.
* **Experiment Tracking:** Managed via MLflow for model versioning and metrics.
* **Orchestration:** Workflow management using Prefect for robust pipeline execution.
* **Environment Management:** High-performance dependency handling using uv.
* **Containerization:** Fully dockerized for seamless deployment across environments.
* **Development:** Built with a Type-Safe approach using Pylance and TypedDict for enhanced code reliability.
